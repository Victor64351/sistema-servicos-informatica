# Sistema de Serviços de Informática Doméstica

Este repositório documenta um sistema pessoal de prestação de serviços
de informática doméstica (desktop), operado com escopo fechado,
preço fixo e baixa frequência.

Não se trata de empresa, startup ou serviço escalável.
O objetivo é organização, controle e execução sem atrito.

Este material é documental.
